# PDF CJK font asset

`NotoSansCJKsc-VF.ttf` is the Simplified Chinese variable TrueType edition of
Noto Sans CJK. It is bundled so every generated Review Notes PDF embeds the same
glyphs on macOS, Linux, CI and future agent runs instead of depending on a
machine-specific Chinese font.

- Upstream: https://github.com/notofonts/noto-cjk/tree/main/Sans/Variable/TTF
- Raw source: https://raw.githubusercontent.com/notofonts/noto-cjk/main/Sans/Variable/TTF/NotoSansCJKsc-VF.ttf
- Retrieved: 2026-07-19
- SHA-256: `990c807e79c25662a5a9ecf7f971baeb2bf2eab9a559e5ecf15cdfdb8561d21f`
- License: SIL Open Font License 1.1, copied in `OFL-NOTO-CJK.txt`

The generator verifies the digest before use. Generated PDFs embed only the
glyph subset required by the current document.
