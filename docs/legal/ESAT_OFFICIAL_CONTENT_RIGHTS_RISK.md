# ESAT 官方内容权利风险评估

**日期：** 2026-07-17
**评估者：** Codex（产品/工程初筛，非法律意见）
**事项：** 下载 UAT-UK ESAT/ENGAA/NSAA 官方文件并转化为满托商业产品内的原生题目、复习资料和邀请码内容
**保密特权：** 否

## 1. 风险描述

UAT-UK 网站条款把网站材料的默认许可限定为个人使用，并要求直接或间接商业使用取得明确书面许可；条款也限制修改下载副本。满托计划把官方 PDF 下载、拆题、重排、解读并放入用于引流和未来收费服务的网站，超出默认个人使用场景。仅在前端隐藏 PDF 或要求邀请码不能解决底层复制、改编和商业使用权问题。

## 2. 背景与当前事实

- 官方 ESAT 资料页已经发现 39 份相关 PDF：2 份核心文件、5 份模块 Guide、2016—2023 ENGAA/NSAA 共 16 份题纸与 16 份答案。
- 当前内部教研库已下载并逐份核验 39 份文件的本地路径、字节数与 SHA-256；`downloadedFiles = 39` 仅表示内部研究副本完整，`publishableFiles = 0` 仍表示没有任何原件可直接公开。
- 产品目标不是简单外链，而是在满托站内提供专业筛选、知识覆盖、在线作答、复盘、Review Notes 和模考，并可能通过邀请码衔接服务。

## 3. 风险分析

### 严重性：4 — High

未经许可发布或改编核心题库，可能造成下架、返工、权利主张和品牌信任损害；内容是产品主要抓手，因此运营影响显著。

### 可能性：4 — Likely

计划中的商业使用、下载、修改和站内发布正是条款明确要求额外许可的行为；如果直接执行，触发条件不是偶发事件，而是产品设计本身。

### 风险分数：16 — RED / Critical

按本项目采用的严重性 × 可能性框架，该事项在取得书面权利前必须作为生产阻断项处理，并升级给负责人和合格法律专业人士复核。

## 4. 加重因素

- 内容将形成公开或邀请码可访问的系统化题库，而非偶发少量引用。
- 题面会被拆分、重排和加入讲解，涉及修改与衍生表达。
- 网站承担满托引流和未来付费解读，存在直接或间接商业用途。
- 前端防下载只能降低批量搬运，不能创造内容权利。

## 5. 缓释因素

- 当前二进制文件只在 Git 忽略、不会进入公开 bundle 的内部教研库中，并没有把 ESAT 来源清单标记为可练习或可发布。
- 专业—模块规则使用事实性要求的本地数据模型，并保存了来源、周期和核验日期。
- 系统已经把 `discovered`、`downloaded` 和 `publishable` 设为三个独立状态。

## 6. 缓释选项

| 方案 | 效果 | 成本/工作量 | 建议 |
| --- | --- | --- | --- |
| 向 UAT-UK 取得覆盖下载、改编、站内展示、商业使用和访问控制的书面许可 | 高 | 中 | 是 |
| 在许可完成前只发布自研题、原创讲义和事实性专业/模块规则 | 高 | 高 | 是，作为并行方案 |
| 只隐藏 PDF 下载按钮或加邀请码后继续使用官方题 | 低 | 低 | 否 |
| 依赖“官网可以下载”推定再发布许可 | 低 | 低 | 否 |

## 7. 建议路径

1. 由满托指定负责人向 UAT-UK 书面确认：允许保存本地副本、拆题/重排、公式与图形在线呈现、中文讲解、邀请码访问、商业引流/收费解读，以及允许使用的年份和署名方式。
2. 由合格法律专业人士审核回复和最终许可文本；许可文件进入非公开权利档案，并在内容 manifest 中记录范围、期限和限制。
3. 在书面许可满足前，自动化可以维护内部研究副本、来源清单和开发导入工具，但不能把原件复制到公开路径，也不能生成包含官方题面的可发布 revision。
4. 同时启动原创 ESAT 模拟题和原创 Review Notes 路线，避免生产进度完全依赖外部许可。

## 8. 残余风险

取得范围明确的书面许可并实施来源、版本、署名和访问控制后，预计可降为 Medium；具体等级取决于许可期限、撤销权、适用材料和商业使用范围。

## 9. 监控与复核

- 每个申请周期更新前复核 UAT-UK 条款和书面许可范围。
- 任何来源域名、发布者、许可措辞、收费模式或访问模式变化时重新评估。
- `publishableFiles` 从 0 变为正数前，必须有权利负责人签字记录和内容审核证据。

## 10. 下一步

1. 满托负责人提供现有书面许可或发起权利申请。
2. 法律专业人士确认许可范围。
3. 工程在许可后以现有 SHA-256 清单为输入，执行 MinerU 解析、逐题人工复核和原生发布流程。
