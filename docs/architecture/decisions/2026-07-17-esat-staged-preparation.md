# 2026-07-17：ESAT 采用专业优先的独立页面流程

**状态：** 已采纳

## 背景

ESAT 的考试模块由申请院校和专业决定。把专业选择、考试结构、课程覆盖和资料入口放在同一长页面，会让用户在尚未确定模块时看到无关内容，也会让导航链接只滚动到说明文字而不是进入可完成的任务。

## 选项

1. 保留单页锚点：实现简单，但不能保存阶段状态，也不能为不同学习进度给出下一步。
2. 采用独立页面：专业定位先确定模块，再进入课程信息、覆盖结果、个人准备、真题和资料页面。

## 决策

采用独立页面，并把顺序固定为：

```text
学校与专业
→ 确定 ESAT 模块（必要时选择允许的组合）
→ 保存到当前设备
→ 填写 A-Level/IAL、IB 或 AP 课程
→ 生成所选模块的课程覆盖初判
→ 进入个人准备、历年真题、模考与笔记
```

首页不再重复展示题量、计时或五个模块说明。导航只提供真实路由，不使用锚点模拟页面。

## 影响

- 新增 ESAT 本地准备计划、课程表单，以及按官方 50 个一级知识单元生成的固定覆盖映射。
- 课程覆盖只回答“课程范围是否提供证据”，不回答实际掌握程度。
- 覆盖页面逐项列出已覆盖、需要确认和需要补充的单元代码及中英文名称；当前不宣称已完成考试局逐条规格等价映射。
- 首页和考试切换器只显示考试对应的专业方向，不显示题量、资料数量或“已整理”等状态文案。
- 真题与模考/笔记入口可联系冰冰获取邀请码，但只能开放已经审核并绑定 entitlement 的内容。
