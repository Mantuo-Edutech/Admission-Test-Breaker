# Manual review evidence

Store one privacy-safe Markdown or JSON review report here for every recorded decision.
Reports should identify the reviewed versions and scope, the checks performed, findings,
resolved issues, limitations and the final recommendation. Use role-based reviewer
references instead of names, email addresses, phone numbers or student-level data.

The decision ledger pins each report by SHA-256. Editing a report after approval causes
the release gate to fail until a new decision is recorded.
